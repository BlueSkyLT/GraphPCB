# Codes for The GraphPCB Dataset


The Full Dataset is hosted at Kaggle [The GraphPCB Dataset](https://www.kaggle.com/datasets/irislan/graphpcb)


# 📁 Folder Structure

```
GraphPCB/
├── Graph-F/
│   ├── graphs/       # Graph data (.pt files)
│   ├── image_dir/    # Original PCB images used to generate the graphs
│   ├── mask_dir/     # Component detection masks
├── Graph-W/
│   ├── graphs/
│   ├── image_dir/
│   ├── mask_dir/
```
